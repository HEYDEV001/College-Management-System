# College-Management-System

A simple Java-based system for managing college information including students, admissions, professors, and subjects.

---

##  Project Objective

To build a basic college management system that handles:
- Student information and records
- Admission data
- Professor/Faculty details
- Subject information

---

##  Project Structure

```
College-Management-System/
├── .mvn/wrapper/           # Maven wrapper
├── src/
│   ├── main/
│   │   ├── java/          # Java source code
│   │   └── resources/     # Configuration files
│   └── test/              # Test files
├── pom.xml                # Maven configuration
├── mvnw                   # Maven script (Linux/Mac)
└── mvnw.cmd              # Maven script (Windows)
```

---

##  Key Features

- **Student Management** - Add, view, and manage student records
- **Admission Management** - Handle admission information
- **Professor Management** - Manage professor/faculty details
- **Subject Management** - Organize subject information

---

##  Technology Stack

- **Language**: Java
- **Build Tool**: Maven
- **Database**: MySQL
- **Framework**: Spring Boot (optional)

---

##  Getting Started

### Prerequisites

- Java JDK 8 or higher
- Maven 3.6.0 or higher
- MySQL database

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/HEYDEV001/College-Management-System.git
   cd College-Management-System
   ```

2. **Build the project**
   ```bash
   mvn clean install
   ```

3. **Run the application**
   ```bash
   mvn spring-boot:run
   ```
